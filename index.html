SPACE COMMUNICATION LOG

HTTP Status Pages

HTTPリクエストを宇宙船から送られる通信信号に見立てた、 宇宙通信エラー風のステータスページ集です。

画像ファイルは使わず、HTMLとCSSだけで近未来的な通信ログの雰囲気を表現しています。
  • 403 Forbidden ⛔ 制限区域へのアクセスが拒否されました。より高い認証レベルが必要です。
  • 404 Not Found 🛰️ 指定された座標にページは存在しません。リクエスト信号は宇宙のどこかで迷子になったようです。
  • 500 Internal Server Error 💥 サーバー基地の内部システムで異常が発生しました。復旧までしばらくお待ちください。
